# MANIL704 PWA complet

Dossier prêt pour GitHub Pages et PWABuilder.

## Installation
1. Envoyer tout le contenu de ce dossier à la racine du dépôt `manil704-IA`.
2. Vérifier que GitHub Pages pointe bien vers la branche principale et le dossier racine.
3. Ouvrir : `https://bouhloujat70.github.io/manil704-IA/`
4. Relancer le test sur PWABuilder.

## Fichiers importants
- `index.html` : interface MANIL704.
- `manifest.json` : manifeste PWA corrigé avec id, icônes et screenshots.
- `sw.js` : service worker pour l'installation PWA et le cache.
- `icons/` : icônes aux bonnes tailles.
- `screenshots/` : captures aux tailles attendues par le manifest.
