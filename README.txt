# MANIL704 PWA

Fichiers prêts pour PWA Builder :
- index.html
- manifest.json
- sw.js
- logo.png
- dossier icons/

Installation / test :
1. Mets tout le dossier sur un hébergement HTTPS.
2. Ouvre l'URL dans https://www.pwabuilder.com/
3. Clique sur "Start" puis vérifie les icônes, le manifest et le service worker.
4. Pour Android, génère le package depuis PWA Builder.

Important : une PWA ne s'installe correctement que depuis HTTPS, pas simplement en ouvrant le fichier HTML localement.
